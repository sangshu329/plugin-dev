# Kite

Kite is flexible layout helper CSS library.

# Demo

See [GitHub Pages](http://hiloki.github.io/kitecss/).

# Browser Support

- Google Chrome (latest)
- Opera (latest)
- Firefox (latest)
- Safari 5+
- Internet Explorer 8+
- Android 2.3+
- Mobile Safari 5+
